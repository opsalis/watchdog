What PingDog monitors

        
Seven check types, including three that nobody else offers. Built for the modern stack — APIs, blockchain nodes, and AI agents.

        

          

            
🌐

            
HTTP / HTTPS

            
Status code, response time, and body match. Catch 5xx errors, wrong content, and slow responses before your users do.

          

          

            
🔌

            
TCP Port

            
Is the port open? How fast does it connect? Monitor databases, mail servers, and raw TCP services.

          

          

            
🔎

            
DNS Resolution

            
Resolution time, resolved IPs, and NXDOMAIN detection. Know when DNS propagation breaks your service.

          

          

            
🔒

            
SSL Certificate

            
Certificate validity and days until expiry. Get alerted 30 days before your cert expires — not after it already has.

          

          

            
⛓️

            
RPC Health

            
Block height, sync status, and staleness detection. Know if your node is stuck or 200 blocks behind without checking manually.

          

          

            
🤖

            
A2A Agent

            
Discovers the agent card, verifies required fields, and pings the agent endpoint. Catch broken AI agents before they fail in production.

          

          

            
🛠️

            
MCP Server

            
Connects via WebSocket, sends a tools/list request, and verifies the tool catalog returns. Know your MCP server is alive and responding.
